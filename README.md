 Basic Network Packet Sniffer (Python + Kali Linux)
 Project Overview

This project is a simple Python-based network packet sniffer developed on Kali Linux.
It captures live network traffic and displays useful packet information such as:

Source IP address
Destination IP address
Protocol type
Payload data

The purpose of this project is to understand how data flows through a network and how common protocols work at packet level.

Objectives

Learn packet capturing fundamentals
Understand IP / TCP / UDP protocol structure
Practice Python networking
Gain hands-on cybersecurity experience
Build a beginner-level security tool

Technologies Used

Python 3
Scapy Library
Kali Linux
Git & GitHub

## Run
sudo python3 sniffer.sy

## Learning Outcome
Network traffic analysis.
